Ed's and Rahul's Tools©

A key component in understanding student conceptions and misconceptions is the analysis of student responses to open-ended questions. 

In the course of building concept inventory instruments, we use common misconceptions in the student's own natural language as distractors. 

 
 

Finding common misconceptions can be a tedious, time-consuming process. 

To facilitate this work, we have developed a web/java-based system (Ed's Tools) that can be used to i) capture student responses, ii) code those responses with respect to any of a number of dimensions (e.g. concepts present, misconceptions present, over-all level of expertise-exhibited, etc.); and then iii) recover specific responses - e.g. all examples of a specific misconception in the studetn's own language.

Ed's Tools greatly speeds the process of analyzing student responses and streamlines concept inventory development.

Rahul's Tools (currently in development) analyzes student responses to various concept inventory instruments and generates a misconception report for both the student, instructor (and us).  Stay tuned, it should be available soon...

If you are interested in using either Ed's or Rahul's tools in your work - let us know.
